## Student Performance Prediction System
### KCA University Academic Analytics & Early Warning Platform

**Project Overview**

The **Student Performance Prediction System** is a web-based academic analytics platform developed to help KCA University monitor, analyze and predict student academic performance using machine learning.

The system is designed to:
* Predict student performance using academic and demographic data.
* Identify at-risk students early.
* Support lecturers and faculty in academic decision-making.
* Improve intervention strategies.
* Reduce dropout rates
* Provide role-based dashboards for students, lecturers and faculty.

This project combines **Machine Learning, Data Analytics and Web Development** into a unified educational intelligence system.

### **🎯 Project Objectives**
**Main Goals**
* Collect and clean student academic datasets
* Train predictive models (Logistic Regression & Decision Tree)
* Build a web-based dashboard for uploads, reports and predictions.
* Generate visual insights for performance analysis
* Support institutional decision-making.

### **👥 User Roles & Functionalities**
**1. Faculty Admin**
* View all student records
* View all lecturer uploads
* Analyze GPA trends
* Filter by unit, lecturer, GPA range
* Export reports (PDF/Excel)
* View attendance insights
* Monitor institutional performance

**2. Lecturer**
* Login securely
* Upload student results via CSV
* View students in assigned units only
* Generate class/unit reports
* Track student performance trends

**3. Student**
* Login securely
* View personal academic performance
* View scores per unit
* Access individual reports
